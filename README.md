# AI based Cattle Analysis and Tracking (Kattlescan ATC)

This project is an AI based system designed to analyze cattle images and classify breeds using deep learning. It uses computer vision techniques along with a CNN model to automate cattle identification and basic body parameter analysis.

## Overview

The system accepts an input image of cattle, applies preprocessing using OpenCV, and then uses a trained convolutional neural network to predict the breed and extract relevant features.

## Features

Cattle breed classification using CNN (MobileNetV2)  
Image preprocessing using OpenCV  
Feature extraction and basic body measurements  
ATC score generation for analysis  
FastAPI backend for handling image input and responses  

## Tech Stack

Python  
TensorFlow and Keras  
OpenCV  
NumPy  
FastAPI  

## Working

1. User uploads an image of cattle  
2. Image is preprocessed using computer vision techniques  
3. Features are extracted from the image  
4. The processed image is passed to the CNN model  
5. The model predicts the breed and generates analysis output  

## Results

Traditional machine learning methods such as Random Forest showed very low accuracy.  
The CNN model using MobileNetV2 improved performance significantly with better classification results.

## How to Run

Install dependencies  

pip install -r requirements.txt  

Run the backend server  

uvicorn main:app --reload  

Open in browser  

http://127.0.0.1:8000/docs  

Use the interface to upload an image and get predictions.
